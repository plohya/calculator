Правила для корректного использования:

1. Для произведения какой-либо операции над числом необиходимо вывести на дисплей необходимое для операции число и после этого нажать на кнопку операции.
(Пример: 169=> sqrt = 13)

2. Для ввода отрицательного числа необходимо путем нажатия кнопок с цифрами вывести их на дисплей, а после этого нажать на кнопку "+-".
(Пример: 9=> +- = -9)

3. Для возведения числа в любую степень необходимо выбрать число, нажать кнопку "^".
(Пример: 2=>^ 2 = 4; 3=>^ 4 = 81)
